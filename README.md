## How to Use

1. **Open the website**: Simply open `index.html` in your web browser
2. **Customize content**: 
   - Edit `index.html` to update your personal information
   - Replace placeholder text with your actual details
   - Update project information and links
   - Add your email, phone, and social media links
3. **Customize styling**: 
   - Edit `styles.css` to change colors, fonts, or layout
   - Modify CSS variables at the top of the file for quick color changes
4. **Add functionality**: 
   - Edit `script.js` to add or modify interactive features
